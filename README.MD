# 🎬 Movie Recommendation Engine

An end-to-end ML data pipeline project built with:

- **Data**: MovieLens 100K dataset
- **Database**: MySQL with Bronze → Silver → Gold layers
- **ML**: Content-Based Filtering using Cosine Similarity
- **Dashboard**: Streamlit

## Features
- Search from 9,700+ movies
- Get instant ML-powered recommendations
- View dataset insights and charts

## Tech Stack
Python · Pandas · Scikit-learn · MySQL · Streamlit
